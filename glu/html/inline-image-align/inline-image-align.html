vertical-align on inline images

baseline (default)

Text before text after — image bottom sits on the baseline, the rest pushes upward and leaves the line-top well above the text cap height.

vertical-align: text-top

Text before text after — image top now aligns with the parent font's ascent (cap height line); the lower portion of the image dips below the text baseline.

vertical-align: top (aliased)

Text before text after — htmlbag maps CSS top to the same resolution as text-top.
