This repository contains ready-to-run examples for the [bagme](https://github.com/boxesandglue/bagme) library.

[Accessible PDF](basic/accessible) | PDF/UA output with automatic structure tagging for screen readers | <a href="basic/accessible"><img src="basic/accessible/result.png" width="200"></a>
[Multipage](basic/multipage) | Automatic page breaks, forced breaks with `page-break-before` | <a href="basic/multipage"><img src="basic/multipage/result.png" width="200"></a>
[Positioned](basic/position) | Manual placement with `OutputAt` using absolute coordinates | <a href="basic/position"><img src="basic/position/result.png" width="200"></a>
[Table](basic/table) | Tables with repeating headers across page breaks | <a href="basic/table"><img src="basic/table/result.png" width="200"></a>
